# Re-ARranged-Master-Mode
**Re-ARranged Pixel Dungeon - In-game Editor**

This repository is dedicated purely to automated builds and contains only the GitHub Actions workflow script (`build.yml`). During compilation, it automatically fetches the core mod logic from the main project and dynamically injects it into the official [Re-ARranged Pixel Dungeon](https://github.com/Hoto-Mocha/Re-ARranged-Pixel-Dungeon) source code.

**Features & Important Notes**
For full documentation on the sandbox editor's features, known limitations (such as game crashes caused by boss floor binding), and save upgrade warnings regarding Mod Items, please refer to the main repository:
👉 **[Shattered-Master-Mode](https://github.com/edward9s/Shattered-Master-Mode)**
